## Git / Commits

**SEMPRE** usar a skill `git-commit` ao commitar qualquer alteração neste projeto.

```
Skill(git-commit)
```

Nunca rodar `git commit` diretamente. A skill garante:
- Branch correta (nunca commitar na `main`)
- Conventional Commits em pt-BR
- Staging inteligente
